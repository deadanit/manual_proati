🔧 Suporte Remoto

ℹ️ Informação: O NIT utiliza o AnyDesk para oferecer suporte técnico remoto às escolas.

💻 AnyDesk

  • Software de acesso remoto
  • Permite suporte à distância
  • Instalado nos computadores administrativos das escolas

🔑 Como funciona

  1. Usuário abre o AnyDesk no computador
  2. Informa o ID do AnyDesk para o NIT
  3. NIT solicita conexão remota
  4. Usuário autoriza o acesso
  5. NIT realiza o suporte necessário
✅ Vantagens: O suporte remoto permite resolver muitos problemas rapidamente, sem necessidade de deslocamento do técnico até a escola.
Caso o Any Desk não esteja instalado no computador, efetue o download clicando aqui.
